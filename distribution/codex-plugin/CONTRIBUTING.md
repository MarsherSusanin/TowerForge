# Contributing

This repository is a generated distribution mirror. Runtime and plugin changes belong in
[Lindforge-Studios/TowerForge](https://github.com/Lindforge-Studios/TowerForge).

Please open feature requests and non-sensitive bug reports in the mirror issue tracker. Source
changes should be proposed against the canonical TowerForge repository. Direct pull requests that
edit generated runtime files cannot be merged because the next publication replaces them.
